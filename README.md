🎯 On The Point Archery Club (Flask Web Application)

On The Point Archery Club is a full-featured web application built using Flask (Python) designed to manage and enhance the experience of an archery training center. The system supports class bookings, event management, memberships, instructor profiles, and equipment rentals in a structured and user-friendly platform.

🚀 Features
🏹 Program and class management (Beginner, Youth, Advanced)
📅 Booking system for classes, private lessons, and events
🎯 Equipment rental management
🏆 Competition and leaderboard tracking
👨‍🏫 Instructor profiles and management portal
👨‍👩‍👧 Parent and youth support features
🧾 Membership system with user dashboards
📣 Community events and announcements
🔔 Notifications for bookings and updates
🧑‍💼 Admin panel for full system control
🛠️ Tech Stack
Backend: Flask (Python)
Frontend: HTML, CSS, Bootstrap
Database: MySQL / SQLite (depending on setup)
Authentication: Flask-Login
UI Design: Mobile-first responsive layout
📌 Project Purpose

This project is developed as a learning and implementation system for designing real-world web applications using Flask. It focuses on role-based access, system design, and scalable architecture for a sports training facility.

🎯 Target Users
Beginners learning archery
Parents enrolling children in classes
Experienced archers seeking advanced training
Community members joining events
Corporate clients organizing team-building activities
Admins and instructors managing operations
📂 Project Status

🚧 Currently under development (UI and backend integration in progress)

📖 Future Improvements
Online payment integration
Real-time notifications
Mobile app version
